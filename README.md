# Safe Spot
## Requirements
- PHP 8.*
- Symfony
- composer

## Run locally
1. Install dependencies
``` shell
composer install
```
2. Run application
``` shell
symfony server --port=8080
``` 
3. Open http://localhost:8080 in browser
